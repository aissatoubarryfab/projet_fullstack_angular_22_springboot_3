# projet_fullstack_angular_22_springboot_3

- Application de gestion de clients developpée avec Spring Boot 3 et Angular 22
- Elle permet d'effectuer les opérations CRUD(création, consultation, modification et suppression) sur les données des clients via une API REST et une interface utilisateur.
# Technologies utilisées 
- Java 17
- Spring Boot 3
- Spring Data JPA
- MySQL
- Angular 22
- TypeScript
- Maven
